## 🚀 Overview

Traditional school transportation systems rely on fixed routes and manual communication, often resulting in:

- Missed student pickups
- Long waiting times
- Inefficient routes
- Increased fuel consumption
- Poor communication between drivers and students

The **Bus Driver Student Tracking System (BDSTS)** is a smart GPS-enabled platform that provides real-time coordination between bus drivers and students. Students can digitally confirm their availability while drivers receive live pickup information, enabling optimized routes and improved transportation efficiency.

## 🚀 How It Works

1. Student opens the app.
2. Student marks themselves **Present**.
3. Student location is uploaded to the cloud.
4. Driver dashboard receives the update instantly.
5. GPS calculates the optimized pickup route.
6. Bus location is shared with students.
7. Student receives ETA notifications.
8. Driver completes pickups efficiently.
