🚌 Bus Driver Student Tracking System

🌐 Live Demo

🔗 https://bustracer.netlify.app/

 📌 Description

The Bus Driver Student Tracking System is a real-time, location-based web application designed to improve coordination between bus drivers and students. It enables students to share their live location and confirm their availability before pickup, allowing drivers to make informed decisions and optimize routes.

🎯 Problem Statement

Traditional student transportation systems face several challenges:

* Missed pickups due to lack of communication
* Uncertainty about student availability
* Increased waiting time for both drivers and students

💡Solution

This system introduces a smart solution where:

* Students share their live location
* Drivers can view all students on a map
* Students receive confirmation requests before pickup
* Drivers skip unnecessary stops based on responses

This improves efficiency, reduces delays, and ensures reliable transportation.

⚙️ Features

* 📍 Real-time student location tracking (prototype simulation)
* 🗺️ Map-based visualization of students
* 🔔 Automated pickup confirmation system
* 👨‍🎓 Student dashboard
* 🚍 Driver dashboard
* ⚡ Dynamic status updates (Confirmed / Skipped / Waiting)

🛠️ Technologies Used

* **Frontend Framework** — React 18 + TypeScript
* **Build Tool** — Vite (with SWC)
* **Styling** — Tailwind CSS + shadcn/ui
* **Animations** — Framer Motion
* **Maps** — Leaflet
* **Routing** — React Router DOM v6
* **State Management / Data Fetching** — TanStack React Query
* **Forms & Validation** — React Hook Form + Zod
* **Backend & Database** — Firebase (planned / optional integration)
* **Charts & Visualization** — Recharts

🏗️ System Architecture

The system consists of two main modules:

1. **Student Interface**

   * Shares location
   * Responds to pickup confirmation

2. **Driver Interface**

   * Views student locations on map
   * Sends confirmation requests
   * Decides pickup route

▶️ How to Run Locally

```bash
git clone https://github.com/your-username/bus-tracking-system.git
cd bus-tracking-system
npm install
npm run dev
```
